# TRex Run
Jogo TRex Run desenvolvido utilizando a Unity.

## Funcionalidades Desenvolvidas
* Movimentação do cenário
* Movimetação do TRex
* Geração de obstáculos
* Marcador de pontos

# Tecnologias Utilizados
* Unity
* C#
